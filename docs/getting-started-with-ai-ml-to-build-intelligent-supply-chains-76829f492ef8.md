# 背景

供应链优化是一个广泛的研究领域。供应链中有大量的用例可以从AI/ML技术的应用中获益。组织往往不知道从何处以及如何开始这方面的工作。供应链高管通常寻找可以投资团队时间和精力的领域（团队已经很紧张），以从这些方法中获得最大价值。在本文中，我们深入探讨了一小部分多样化的用例，这些用例可以作为供应链组织涉足AI/ML的起点。供应链领导者可以期待从这些应用中获得显著的成本和效率提升。

我们将供应链管理分为五个组成部分：计划、采购、制造、交付、逆向物流，并概述了*需求感知*、*供应商细分*、*设备故障预测*、*交付时间预测*、*客户退货预测*等用例，这些用例对应于这五个组成部分。应用经济价值、可操作性、结果可解释性、洞察行动性等标准…
