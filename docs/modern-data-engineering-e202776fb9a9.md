现代数据生态系统不断演进，新的数据工具时不时出现。在这篇文章中，我想讨论影响数据工程师的关键问题。我们将探讨如何利用这些知识来推动高级分析管道和卓越运营。

## 我想讨论一些热门的数据工程问题：

+   现代数据工程（DE）。它是什么？

+   你的DE是否足够好，以支持高级数据管道和商业智能（BI）？

+   你的数据管道是否高效？

+   从技术角度来看，实现卓越运营需要什么？

回到十月份，我曾写过关于数据工程师崛起的文章，讨论了这一角色、其挑战、责任、日常工作以及如何在这一领域获得成功。数据工程的格局不断变化，但主要趋势似乎保持不变。
