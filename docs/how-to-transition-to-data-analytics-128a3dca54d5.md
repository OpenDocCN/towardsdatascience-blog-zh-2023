# 进入数据分析的两种方式

实际上，有两种方式可以进入数据分析领域：

+   (1) 完全自学：然后将分析技能与当前领域知识（无论是早期学习还是工作经验）巧妙结合，以获得竞争优势；

+   (2) 参加数据分析学位课程或培训班。

在这篇博客文章中，我们将主要关注**自学**的**路径**，目标是成为数据分析师，这是许多其他数据科学角色的良好起点。这也是我六年前的起点，当时我对医疗分析非常感兴趣，所以这是我开始的领域。
