# 介绍

当你在数据科学家职业生涯中工作时，很容易将注意力集中在*硬技能*上。你可能想要学习一种新的 ML 算法，比如带有非线性核的 SVM，一种新的软件技术，比如 MLflow，或者一种新的 AI 趋势，比如 ChatGPT。

这些技能学习起来很舒适，因为**衡量成功非常简单**。以 MLflow 为例。你可能会首先了解 MLflow 可以为你的 ML 生命周期提供什么。你学习模型工件、ML 项目结构和模型注册。你完成了一个课程，花了几个小时阅读用户指南，甚至在现实项目中实施了它。太棒了！在做完这些之后，你可以自信地说你了解了一些 MLflow，并可以将其作为技能添加到你的简历中。
