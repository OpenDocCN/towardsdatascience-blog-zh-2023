# 介绍

在 Python 中，字符串是不可变的字符序列，用于处理文本数据。

你应该了解的字符串关键知识点包括：

+   有序的

+   不可变的

+   可迭代的

学习每种编程语言中的数据类型对于理解代码和程序至关重要。
