# 数据工程与编程技能

当我们想到数据工程时，通常首先想到的编程技能是 SQL 和可能的 Python。SQL 是一种用于查询数据的知名语言，深深根植于数据和管道的世界中。另一方面，Python 在数据科学中变得非常强大，并且现在在不断发展的数据工程领域中也开始崭露头角。但是，这种普遍看法准确吗？SQL 和 Python 真的对数据工程师来说是最重要的编程技能吗？在本文中，我将分享我的经验，旨在帮助年轻的专业人士找出最佳技能，从而最大限度地利用他们的时间和精力。

# 为什么选择 Java 和 Scala？
