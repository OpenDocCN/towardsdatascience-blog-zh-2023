变换器在人工智能领域主导了六年，已在所有人工智能子领域中取得了最先进的成果。从[自然语言处理 (NLP)](https://en.wikipedia.org/wiki/Natural_language_processing)到计算机视觉，再到声音和图形，专门的变换器都表现出色。

+   这个领域还会持续多久？

+   变换器真的是最好的架构吗？

+   它会在不久的将来被取代吗？

+   对其主导地位的威胁是什么？

这篇文章试图回答这些问题。从为什么变压器如此成功以及哪些因素使其在如此多不同领域中建立起自己的地位开始，我们将分析它是否仍然具有无可争议的主导地位，哪些因素威胁到它的至高无上的地位，以及是否存在潜在的竞争者。
