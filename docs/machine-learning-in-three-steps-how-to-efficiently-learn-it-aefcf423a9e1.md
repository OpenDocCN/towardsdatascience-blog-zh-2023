我观察到有两种极端的方法用于有志成为数据科学家的学习机器学习算法。第一种方法涉及学习**所有的细节**并从零开始实现这些算法，以获得真正的掌握。另一方面，第二种方法假设计算机会“自我学习”，从而使个人学习算法变得不必要。这导致一些人**仅仅**依赖诸如 lazypredict 这样的工具。

在学习机器学习算法时，采取一种介于两种极端之间的方法是现实的。然而，问题仍然存在，应该从哪里开始？在这篇文章中，我将把机器学习算法分为三类，并提供我谦虚的意见，说明该从什么开始，以及可以跳过哪些内容。

# 机器学习算法的复杂性

机器学习入门可能会因为众多可用的算法而感到不知所措。线性回归、支持向量机（SVM）、梯度下降、梯度提升、决策树、LASSO、岭回归、网格搜索等，都是在面对这个问题时浮现在脑海中的一些算法。
