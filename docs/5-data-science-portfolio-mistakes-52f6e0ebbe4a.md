在尝试作为数据科学家获得工作时，重要的不仅仅是你知道什么，还包括**你展示了什么**。换句话说，有效地传达你的专业知识和过往工作是过程中的关键部分。

实现这一目标的最佳方法之一是通过一个作品集网站。作品集可以提升你的信誉，并使你的工作随时可供客户和招聘经理查看。

然而，并非所有作品集都是相同的。有一些关键错误会**将差作品集与好作品集区分开来**。在本文中，我将详细分析 5 个错误，这些错误会确保你的作品集永远无法让你获得工作。

# **错误 1: 不要制作作品集网站**

> 最糟糕的作品集就是没有作品集。

由于大多数人没有作品集，拥有*一些东西*将带来大部分好处，并让你在其他候选人中脱颖而出。

然而，如果你从未制作过网站，你可能会犹豫。你可能担心这会花费金钱，或者你需要花费数小时学习如何编写 HTML 和 CSS 代码。

我也曾经经历过这种情况。有趣的是，我花了 8 个月才制作出我的第一个作品集，这不是因为成本或时间投入巨大，而是因为*我*…
