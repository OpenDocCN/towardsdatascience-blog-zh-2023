# 这篇文章讲了什么？

“你为什么认为负责任的生成式人工智能（GenAI）是重要且紧迫的？”这是政策制定者、研究人员、记者和关心的公民们今天提出的问题。GenAI 的迅速发展引起了公众的想象，但也提出了紧迫的伦理问题。像 ChatGPT、Bard 和 Stable Diffusion 这样的模型展示了这项技术的创造潜力——但在错误的手中，这些相同的能力可能会以空前的规模促进虚假信息和操控。与以前的技术不同，GenAI 使得创建高度个性化、特定上下文的合成媒体成为可能，这些媒体难以被验证为虚假。这带来了新的社会风险和复杂的治理挑战。

在这篇博客文章中，我将深入探讨四个方面（规模与速度、个性化、来源、扩散），这些方面将新一代生成式人工智能（GenAI）与过去的时代区分开来，并强调为什么现在是研究人工智能伦理和负责任使用的正确时机。在本文中，我旨在通过突显关键方面来回答“为什么是现在？”这个问题。可能的解决方案将会被探索……
